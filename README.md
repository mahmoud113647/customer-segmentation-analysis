# Mall Customer Segmentation

## Overview
This project analyzes mall customer data using Python and applies K-Means clustering for customer segmentation.

The analysis includes data preprocessing, exploratory data analysis, feature scaling, correlation analysis, and clustering to identify customer groups based on annual income and spending score.

## Dataset
- Mall_Customers.csv

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Steps
- Loaded and explored the dataset
- Removed unnecessary columns
- Analyzed data types and summary statistics
- Visualized distributions and relationships using plots
- Generated correlation heatmap
- Scaled numerical features using StandardScaler
- Applied Elbow Method to determine optimal number of clusters
- Performed K-Means clustering
- Visualized customer segments

## Key Techniques
- Data Analysis
- Data Visualization
- Feature Scaling
- K-Means Clustering
- Customer Segmentation

## Output
The final result is a customer segmentation model that groups customers based on spending behavior and annual income.
